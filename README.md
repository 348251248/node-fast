# Node backend with fastify

- framework: Fastify
- database: MongoDB + Mongoose
- API: Swagger(Open API)

# Command

## To start server with Nodemon

```sh
yarn start
```

### To view api documentation

- start server
- go to localhost:3000/documentation

## Run test

```sh
yarn test
```
